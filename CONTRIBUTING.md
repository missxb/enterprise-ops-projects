# 贡献指南

欢迎提Issue讨论技术问题，欢迎PR补充实际使用经验。

## 如何贡献

1. Fork 本仓库
2. 创建特性分支 (`git checkout -b feature/your-feature`)
3. 提交更改 (`git commit -m 'Add some feature'`)
4. 推送到分支 (`git push origin feature/your-feature`)
5. 创建 Pull Request

## 内容规范

- 脚本文件放 `scripts/` 目录
- 配置文件放 `configs/` 目录
- 故障案例请包含：现象、排查过程、根因、解决方案、预防措施
- 所有密码使用占位符，不要提交真实凭据

## 问题反馈

如有问题或建议，请 [提交 Issue](https://github.com/missxb/enterprise-ops-projects/issues)。
